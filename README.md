# 🛡️ SOC Home Lab

## Overview

This repository documents my hands-on journey of building a Security Operations Center (SOC) Home Lab. The objective is to develop practical Blue Team skills through real-world cybersecurity labs involving Linux administration, log analysis, attack simulation, SIEM deployment, and incident investigation.

The lab environment is built using VMware Workstation with Ubuntu and Kali Linux virtual machines. Each lab focuses on a specific security concept and includes detailed documentation, screenshots, commands, observations, and lessons learned.

---

## Lab Environment

| Component        | Technology             |
