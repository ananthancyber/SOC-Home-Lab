# 🛡️ SOC Home Lab

## Overview

This repository documents my hands-on journey in building a Security Operations Center (SOC) Home Lab using Ubuntu, Kali Linux, VMware Workstation, and Wazuh SIEM.

The objective is to simulate cyber attacks, collect and analyze logs, investigate security incidents, and develop practical Blue Team skills.

---

## Lab Environment

| Component | Technology |
|-----------|------------|
| Hypervisor | VMware Workstation |
| Attacker | Kali Linux |
| Target | Ubuntu 24.04 XFCE |
| Future SIEM | Wazuh |
| Log Analysis | journalctl |
